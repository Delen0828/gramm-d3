How to Interact

  • Hover: Move your mouse over data points to see tooltips with exact values
  • Click: Click on data points to select them (selected points turn orange)
  • Multi-select: Hold Ctrl/Cmd while clicking to select multiple points
  • Clear Selection: Use the "Clear Selection" button below
  • Export Data: Use the "Export Data" button to see the JSON output
